# IPTV плейлист с 29 каналами телеканала Триколор Спорт

## Подключение плейлиста
Просто скопируйте ссылку ниже и вставьте в поле для ввода адреса вашего проигрывателя.

В медиапроигрывателе [VLC media player](https://www.videolan.org/vlc/) выберите: "Медиа" -> "Открыть URL..." (или комбинацией клавиш Ctrl+N) и в поле под строкой "Введите сетевой адрес:" вставьте скопированную ссылку. Нажмите "Воспроизвести".

В медиапроигрывателе [IP-TV Player](https://borpas.info/iptvplayer): В правом списке каналов находим сверху три точки. Меню Списки - Добавить. В поле Адрес списка каналов добавляем ссылку на плейлист.

### Ссылка на плейлист формата m3u8:
```
https://sx007.github.io/IPTV/Tricolor_Sport.m3u8
```
Данный формат **m3u8** плейлиста можно открыть в проигрывателе VLC media player, а так же в IP-TV Player.

### Ссылка на плейлист формата m3u:
```

```
Данный формат **m3u** плейлиста можно открыть в проигрывателе VLC media player, а так же в IP-TV Player.

### Ссылка на плейлист формата xspf:
```
https://sx007.github.io/IPTV/Tricolor_Sport.xspf
```

Данный формат **xspf** плейлиста можно открыть в проигрывателе VLC media player.
